# 可步行性数据集构建工具

## 项目简介

可步行性数据集构建工具是一个基于Streamlit开发的Web应用，用于从大模型输出中提取问题归因结果，并根据Excel表格匹配对应的优化策略，同时支持数据集构建和JSONL处理。该工具主要用于城市规划领域的可步行性分析，帮助研究人员和规划师高效处理和分析街景图像数据。

## 功能特点

- ✅ **优化策略提取**：从大模型输出中提取问题归因结果，并根据Excel表格匹配对应的优化策略
- ✅ **JSONL处理**：将JSONL文件转换为JSON文件，并支持进一步转换为Excel和结构化Excel文件
- ✅ **大模型数据集构建**：构建多模态或纯文本大模型数据集，支持自定义提示词和多图片输入
- ✅ **命令行记录**：记录和管理命令行代码，支持添加、编辑、删除、搜索和导出记录
- ✅ **预设参数管理**：支持保存、加载和导入预设参数，提高工作效率
- ✅ **预览功能**：在生成数据集前可以预览结果，确保数据集质量
- ✅ **模块化设计**：代码结构清晰，便于维护和扩展

## 安装说明

### 环境要求

- Python 3.8+ 
- Streamlit
- Pandas
- OpenPyXL

### 安装步骤

1. 克隆仓库

```bash
git clone https://github.com/chenyiwang325-droid/walkability-dataset-tool.git
cd walkability-dataset-tool
```

2. 安装依赖

```bash
pip install -r requirements.txt
```

3. 运行应用

```bash
streamlit run app.py
```

4. 在浏览器中访问

```
http://localhost:8501
```

## 使用指南

### 1. 优化策略提取

1. 上传大模型输出文件（JSON、JSONL或Excel）
2. 可选：上传自定义优化策略表格
3. 配置高级设置（输出目录、文件名前缀等）
4. 点击"开始处理"按钮
5. 查看处理结果或下载结果文件

### 2. JSONL处理

1. 配置处理步骤和模式
2. 上传JSONL文件
3. 点击"开始处理JSONL文件"按钮
4. 查看处理结果或下载生成的文件

### 3. 大模型数据集构建

#### 3.1 核心配置

- **图片目录**：设置图片所在的目录路径
- **图片路径前缀**：图片路径的前缀，用于生成最终的图片URL

#### 3.2 模型配置

- **模型类型**：选择模型类型（multimodal或llm）
- **数据集类型**：选择数据集类型（test或train）
- **启用多图片输入**：是否需要在单条数据中添加多张图片
  - 如果启用，可设置最大图片数

#### 3.3 多图片处理配置（仅在启用多图片输入时显示）

- **处理多块图片**：是否处理多个街区的图片，每个街区作为一组
- **按子目录分组**：是否按子目录分组图片，每个子目录作为一组

#### 3.4 内容提取配置

- **启用内容提取**：是否从数据源中提取内容进行组合
  - 如果启用，需选择数据源（excel或json）
  - 根据选择的数据源，配置相应的提取参数

#### 3.5 输出配置

- **输出文件路径**：设置输出文件的路径，支持{street_number}和{dataset_type}变量
- **在输出中包含标签**：是否在输出结果中新增label字段

#### 3.6 提示词组合配置

- **系统提示词**：系统提示词，用于指导模型生成内容
- **用户提示词默认内容**：用户提示词的默认内容
- **提示词组合**：只有启用内容提取时显示
  - 用户提示词前缀
  - 用户提示词后缀
  - 提取内容前缀、后缀和分隔符（仅在启用多图片输入时显示）

#### 3.7 预览和构建

- **预览数据集**：在生成数据集前预览结果
- **开始构建数据集**：生成最终的数据集文件

### 4. 命令行记录

1. 添加新命令行记录
2. 查看、编辑或删除已有记录
3. 搜索记录
4. 导出记录

## 功能模块说明

### 1. 优化策略提取

该模块用于从大模型输出中提取问题归因结果，并根据Excel表格匹配对应的优化策略。支持处理JSON、JSONL和Excel格式的输入文件，输出JSON、Markdown和Excel格式的结果。

### 2. JSONL处理

该模块用于将JSONL文件转换为JSON文件，并支持进一步转换为Excel和结构化Excel文件。支持三种处理模式：
- 仅JSONL转JSON
- JSONL转JSON+JSON转Excel
- 完整流程

### 3. 大模型数据集构建

该模块用于构建大模型数据集，支持多模态和纯文本模型。主要功能包括：
- 支持多图片输入
- 支持从Excel或JSON数据源提取内容
- 支持自定义提示词
- 支持预设参数管理
- 支持数据集预览

### 4. 命令行记录

该模块用于记录和管理命令行代码，方便用户保存和复用常用命令。支持添加、编辑、删除、搜索和导出记录。

## 输出结果

### 优化策略提取

输出结果包括：
- JSON文件：包含提取的问题归因结果和匹配的优化策略
- Markdown文件：格式化的问题归因及优化策略
- Excel文件：结构化的优化策略表格

### JSONL处理

输出结果包括：
- JSON文件：转换后的JSON格式数据
- Excel文件：转换后的Excel格式数据
- 结构化Excel文件：根据特定格式处理后的Excel数据

### 大模型数据集构建

输出结果为JSON格式的数据集文件，每条数据包含：
- messages：包含系统提示词、用户提示词和助手响应（如果是train模式）
- images：图片路径列表（如果是multimodal模型）
- label：标签列表（如果启用）

## 注意事项

1. 建议使用Python 3.8及以上版本
2. 处理大型文件时，可能需要较长时间，请耐心等待
3. 建议将图片文件和数据文件放在同一目录下，便于管理
4. 在使用预设参数时，确保预设参数与当前任务匹配
5. 预览功能仅显示前3条数据，用于快速查看数据集格式
